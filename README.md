# AM Consultoria - Sistema de Gestão da Informação

Sistema desenvolvido para auxiliar a organização operacional da AM Consultoria, empresa voltada exclusivamente para instituições ligadas à Economia do Cuidado.

O projeto foi criado com o objetivo de centralizar informações relacionadas a clientes, visitas, contratos e pendências operacionais, reduzindo a fragmentação de registros e melhorando o acompanhamento das atividades da consultoria.

---

# Sobre o Projeto

Durante a rotina da consultoria, diversas informações importantes ficavam distribuídas entre:
- planilhas;
- anotações físicas;
- aplicativos de mensagens;
- arquivos separados.

Essa fragmentação dificultava:
- recuperação rápida de informações;
- acompanhamento das pendências;
- organização financeira;
- continuidade dos atendimentos.

O sistema foi desenvolvido justamente para solucionar esses problemas por meio de uma aplicação desktop simples, intuitiva e funcional.

---

# Funcionalidades

- Cadastro de clientes
- Registro de visitas
- Controle de pendências
- Organização contratual
- Gerenciamento financeiro
- Histórico de atendimentos
- Centralização das informações
- Interface gráfica intuitiva
